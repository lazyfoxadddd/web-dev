Frontend Development

Frontend development is the development of the GUI of website, through the use of HTML, CSS and JS, so that users can view and interact with that website.

  • HTML : The Hypertext Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.


  • CSS :Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.


    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)


  • JAVASCRIPT :Javascript is often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web.

    1. Vanilla JS

    2. jQuery (Library)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. Typescript (Preprocessor)
